# Responsive Layout

MoodMeal AI uses Tailwind CSS responsive breakpoints:

- `sm:` — small screens (640px+)
- `md:` — tablets (768px+)
- `lg:` — desktops (1024px+)

The navbar stacks vertically on mobile and goes horizontal on larger screens.
Meal cards use a 1-column grid on mobile, 2 on tablet, and 3 on desktop.
